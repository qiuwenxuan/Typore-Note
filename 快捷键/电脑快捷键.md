# 电脑快捷键

| windows+L     | 电脑锁屏         |
| ------------- | ---------------- |
| windows+W     | 关闭当前页面     |
| windows+Table | 打开多个窗口     |
| windows+E     | 打开文件管理器   |
| windows+D     | 切回桌面         |
| Ctrl+W        | 关闭所有页面     |
| Ctrl+Shirft+T | 恢复刚关闭的网页 |
| windows+V     | 剪切板           |
